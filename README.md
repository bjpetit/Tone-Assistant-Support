# Tone Assistant Support

This repository hosts the public support site for the Tone Assistant app as a plain static GitHub Pages site.

## GitHub Pages setup

1. Open **Settings → Pages** in this repository.
2. Set **Source** to deploy from the `main` branch and the `/ (root)` folder.
3. Save the settings.

The existing homepage is `index.html` at the repository root (with existing `styles.css` for styling).

Your support site will be published at:

`https://bjpetit.github.io/Tone-Assistant-Support/`
